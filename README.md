# Color Guess

# Contexto
Esse site propõe um desafio ao usuário que é adivinhar a cor que corresponde ao código RGB mostrado na tela. A pontuação é atualizada conforme os acertos.

# Habilidades Desenvolvidas

- Manipular o DOM.
- Manipular o Javascript.
- Manipular o CSS.
